Asp.net core web api based job finder web app.

People sometimes need a daily employee in any sector such as clothes selling. Important fact here is it is only needed for one day. This app allows to employers to find daily employees for their need. It is a location-based app. Employer will register on app then some field. User will access to system and will see the nearest job.
